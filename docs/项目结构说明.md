# 项目结构说明

> 最后更新：2026-06-21

---

## 总览

```
tianfu_daka/
├── backend/           # Python FastAPI 后端（端口 8011）
├── h5_fronted/        # React H5 前端（旧版，将替换为 Vue）
├── vue_fronted/       # 新版 UniApp Vue 前端（开发中）
│   ├── tests_fronted/ # 设计预览页（给老板看的原型）
│   └── src/           # 正式 UniApp 代码（待迁移）
├── docs/              # 项目文档
├── scripts/           # 启动/停止脚本（Windows + Mac + Linux）
├── tests/             # 集成测试
├── logs/              # 运行日志（自动生成）
└── .venv/             # Python 虚拟环境
```

---

## 后端 `backend/`

| 目录/文件 | 说明 |
|------|------|
| `main.py` | FastAPI 入口，注册路由和 CORS |
| `requirements.txt` | pip 依赖 |
| `.env` / `.env.example` | 环境变量（API Key 等） |
| `config/` | YAML 配置文件 |
| `config/settings.yaml` | 服务器端口、上游 RAG URL、模型配置 |
| `config/questions.yaml` | 105 道题目数据 |
| `config/integration.yaml` | 外部依赖端点状态 |
| `config/loader.py` | YAML 加载器 + 环境变量覆盖 |
| `app/api/` | API 路由层 |
| `app/api/talent.py` | 天赋测试相关接口 |
| `app/api/chat.py` | AI 对话代理接口 |
| `app/api/health.py` | 健康检查接口 |
| `app/models/` | Pydantic 请求/响应模型 |
| `app/services/` | 业务逻辑层 |
| `app/services/jnao_client.py` | 调用外部 JNAO API |
| `app/services/ai_proxy_real.py` | 代理 tianfu_rag AI 对话 |
| `app/services/interfaces.py` | AI 代理接口协议 |
| `app/core/` | 核心工具 |
| `app/core/scoring.py` | 本地评分引擎 |
| `app/mock/` | Mock 服务（降级用） |
| `tests/` | 后端单元测试（pytest） |

| 目录/文件 | 说明 |

| `src/data/` | 题目数据 + AI 提示文案 |
| `src/lib/` | 工具函数（洗牌、编码、存储） |

| `src/projects/talent/` | 天赋测试状态机 |
| `public/天赋/logos/` | 5 种天赋类型 Logo |

## 脚本 `scripts/`

| 文件 | 系统 | 说明 |
|------|------|------|
| `start_all.bat` | Windows | 一键启动后端+前端 |
| `start_all.sh` | Mac/Linux | 一键启动后端+前端 |
| `start_all.ps1` | Windows | PowerShell 版一键启动 |
| `start_backend.bat` / `.sh` | Win/Mac | 仅启动后端 |
| `start_frontend.bat` / `.sh` | Win/Mac | 仅启动前端 |
| `stop_all.bat` / `.ps1` | Windows | 停止所有服务 |
| `kill_port.ps1` | Windows | 清理指定端口占用 |
| `test_child_flow.py` | — | 儿童流程测试脚本 |
