# 项目结构说明

> 最后更新：2026-07-02

---

## 总览

```
tianfu_daka/
├── backend/           # Python FastAPI 后端（端口 8012）
├── vue_fronted/       # UniApp Vue 前端（主前端）
│   ├── src/           # 正式页面（talent / training / qa / growth / login / report）
│   └── tests/         # 前端测试（vitest，3 文件 89 用例）
├── docs/              # 项目文档
├── scripts/           # 启动/停止脚本（Windows + Mac + Linux）
├── tests/             # 发烟测试 + 集成测试
├── logs/              # 运行日志（自动生成）
└── .venv/             # Python 虚拟环境
```

## 后端 `backend/`

| 目录/文件 | 说明 |
|------|------|
| `main.py` | FastAPI 入口，注册路由和 CORS |
| `requirements.txt` | pip 依赖 |
| `.env` / `.env.example` | 环境变量（豆包 API Key 等） |
| `config/` | YAML 配置文件（5 个） |
| `config/settings.yaml` | 服务器端口（8012）、模型配置 |
| `config/training_curriculum.yaml` | 时长公式 / 技能分类 / 选修规则（🔄 v2.0 重构） |
| `config/training_tier_thresholds.yaml` | 技能 Tier×学段→达标阈值（🆕 v2.0） |
| `config/questions.yaml` | 天赋测评题目（105 题） |
| `config/integration.yaml` | 外部 API 对接状态 |
| `config/loader.py` | YAML 加载器 + 环境变量覆盖 |
| `app/api/` | API 路由层（14 个路由模块） |
| `app/api/training.py` | 训练方案 + 打卡 + 窗口 |
| `app/api/talent.py` | 天赋测评 + 报告 |
| `app/api/qa.py` | 学科答疑 + 会话管理 |
| `app/api/growth.py` | 成长里程碑 |
| `app/api/guide.py` | 首页引导对话 |
| `app/api/user.py` | 用户注册/登录/个人信息 |
| `app/api/parent.py` | 家长中心（孩子管理/名额查询） |
| `app/api/auth.py` | 统一认证（家长密码/孩子密码/旧手机号兼容） |
| `app/schemas/` | Pydantic 请求/响应模型 |
| `app/services/` | 业务逻辑层（45+ 个服务模块） |
| `app/services/training_service.py` | 训练核心服务（42KB） |
| `app/services/training_schedule_service.py` | 排课调度引擎（🔄 v2.0 重写） |
| `app/services/training_formula_engine.py` | 时长→技能组合展开（🆕 v2.0） |
| `app/services/training_mastery.py` | 连续3次达标判定→Tier晋级（🔄 v2.0 重写） |
| `app/services/training_elective_service.py` | 选修弹窗管理（🆕 v2.0） |
| `app/services/training_day.py` | 训练日边界（凌晨4点） |
| `app/services/training_carryover.py` | 昨日未完成→今日按技能续训（🔄 v2.0） |
| `app/services/training_day_settlement.py` | 训练日结算（🔄 v2.0 去除主线推进） |
| `app/services/child_training_state.py` | 状态机（多技能 Tier + OSS + 连续计数，🔄 v2.0 重写） |
| `app/services/onboarding_service.py` | 老学员引导落库 |
| `app/services/parent_service.py` | 家长中心（孩子 CRUD + 名额管理） |
| `app/services/dev_clock.py` | 开发用模拟时钟 |
| `app/services/` **v2.0 废弃** | `training_block_builder.py`、`training_duration_pack.py`、`training_curriculum_router.py`、`training_curriculum_scheduler.py`、`training_route_llm.py`、`training_route_context.py`、`training_optional_service.py` |
| `app/db/` | 数据库模型 + 迁移 |
| `app/db/models.py` | ORM 模型（7 张核心表） |
| `app/db/migrate.py` | 轻量 schema 补丁 |
| `app/agents/` | AI Agent 层（人设 + 路由 + 记忆） |
| `tests/` | 后端测试（pytest，38 文件 188 用例） |

## 脚本 `scripts/`

| 文件 | 系统 | 说明 |
|------|------|------|
| `start_all.bat` | Windows | 一键启动后端+前端 |
| `start_all.sh` | Mac/Linux | 一键启动后端+前端 |
| `start_all.ps1` | Windows | PowerShell 版一键启动 |
| `start_backend.bat` / `.sh` | Win/Mac | 仅启动后端 |
| `start_frontend.bat` / `.sh` | Win/Mac | 仅启动前端 |
| `stop_all.bat` / `.ps1` | Windows | 停止所有服务 |
| `kill_port.ps1` | Windows | 清理指定端口占用 |
| `test_child_flow.py` | — | 儿童流程测试脚本 |
| `clear_dev_data.py` | — | 清空开发数据（保留 content_item） |
| `migrate_state_v2.py` | — | 训练状态 v1→v2 数据迁移 |
| `reset_local_data.ps1` / `.py` | Win/通用 | 重置本地数据 |
