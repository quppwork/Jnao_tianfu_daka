# PR: JNAO 天赋成长平台 — 前端完整实现 + 后端 API 清单

## 概述

完成学生端四大核心模块的前端页面，搭建完整的项目基础设施，定义后端全部待开发 API。

---

## 新增前端页面

### 🧭 天赋测试 (`pages/talent/`)
- 大门页 → 年龄确认 → 确认开始 → 35 题答题（60 秒倒计时，超时原地等待）
- 撤销上一题（5 秒窗口）+ 答题反馈（随机 ack + 里程碑提示）
- 完成页动画 → 生成报告 → 后端 JNAO API → 报告页
- 报告页：10 区块（Hero/统计/特质/雷达图 SVG/情绪仪表/建议/黄金建议）
- 历史报告查看（localStorage 持久化）

### 📅 今日训练 (`pages/training/`)
- 赛博朋克深色主题 + 白色系适配
- augmented-ui 折角边框特效
- 12 能力 4×3 宫格（矩阵扫描入场动画）
- 多选打卡表单（扫描线展开/收缩消失动画）
- 文件上传（图片/视频预览）+ 文字备注
- 训练时段选择（时分选择器，时段外禁止播放）
- 开发者模式开关（绕过时段限制）
- 6 档训练配合度打分
- 视频/音频弹窗播放器

### 💬 学科答疑 (`pages/qa/`)
- AI 对话界面（头像呼吸脉冲动画）
- 豆包 API 接入（doubao-seed-2-0-lite）
- 暗白双主题适配

### 🏆 成长里程碑 (`pages/growth/`)
- 6 枚荣誉徽章（已获/未获）
- 成长时间线（完成/未来虚化）
- 分享卡片占位
