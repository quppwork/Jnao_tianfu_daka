# JNAO AI 产品规格

> 最后更新：2026-06-21  
> 版本：v2.1

---

## 一、产品定位

JNAO 是一个面向 K12 学生与家长的 **天赋成长综合平台**。

```
┌──────────┬──────────┬──────────┬──────────┐
│ 天赋测试  │ 今日训练  │ 知识答题  │ 成长里程碑 │
│ 发现天赋  │ 每日打卡  │ AI 辅导   │ 追踪成长  │
└──────────┴──────────┴──────────┴──────────┘
```

**闭环链路：** 测天赋 → 定等级 → 每日训练 → AI 答疑 → 看成长 → 社交分享

## 二、角色体系

```
┌────────────────────────────────────────────────────┐
│  C端（学生端）— 当前开发                              │
│  ┌──────────┬──────────┬──────────┬──────────┐    │
│  │ 天赋测试  │ 今日训练  │ 知识答题  │ 成长里程碑 │    │
│  └──────────┴──────────┴──────────┴──────────┘    │
├────────────────────────────────────────────────────┤
│  B端（导师端/家长端）— 后续开发                       │
│  ┌──────────────────────────────┐                 │
│  │  学习监督（提醒+打分+干预方案）│                 │
│  └──────────────────────────────┘                 │
└────────────────────────────────────────────────────┘
```

| 角色 | 说明 | 优先级 |
|------|------|--------|
| **学生** | 使用四个模块 | 🥇 当前 |
| **家长** | 与学生共用账户，可绑定多个学生 | 🥈 后续 |
| **导师（公司）** | 监督+打分+营销指导 | 🥈 后续 |

### 学生端（当前开发范围）

- 一个学生一个账户
- 注册时填写：年级、成绩水平、训练经历等（字段清单待老板确认）
- 首次使用：天赋测评 → 获得天赋类型 → 确定训练等级

### 家长端（后续）

- 与学生共用账户
- 一个家长可绑定多个学生（每多一个单独购买）
- 首页可切换子女
- 查看各子女的训练数据与报告

## 三、四大模块

### 模块一：天赋测试

| 项目 | 说明 |
|------|------|
| 功能 | AI 驱动的天赋测评 |
| 题目 | 35 道选择题（是/否），约 8 分钟 |
| 类型 | 成人版、儿童版 + 备用卷 |
| 输出 | 五维雷达图 + 情绪仪表 + 天赋解读 + 发展建议 |
| 天赋类型 | 思者 / 赢者 / 德者 / 行者 / 学者（+ 迷者，仅儿童） |
| 后端 | JNAO 外部 API（m.jnao.com）代理 |
| 详情 | [天赋测试.md](功能模块文档/天赋测试.md) |

### 模块二：今日训练（学员指导）

| 项目 | 说明 |
|------|------|
| 功能 | 根据等级提供每日训练方案，打卡解锁 |
| 内容 | 视频训练 + 音频训练 + 验收标准，老板定义 |
| 等级 | 老板定义等级体系，训练内容随等级变化 |
| 跳级 | 老学员填自评表 → 计算等级 → 跳过新手阶段 |
| 报告 | 数据分析 → 生成报告 + 打卡条 + 推荐课程 |
| 详情 | [今日训练.md](功能模块文档/今日训练.md) |

### 模块三：知识答题（学科答疑）

| 项目 | 说明 |
|------|------|
| 功能 | AI 一对一辅导，结合天赋类型个性化回答 |
| 输入 | 语音输入（火山引擎 ASR）+ 文字输入 + 拍照发题 |
| 输出 | 文字回答 + 语音播报（火山引擎 TTS，张宇老师声音） |
| 个性化 | 根据天赋类型注入 AI 提示词（方案A系统提示词 + 方案B轻量上下文） |
| 天赋咨询 | 测评完成后解答天赋相关问题（防扒策略：引导到每日个性化训练） |
| 拍照讲题 | 拍照 → AI 识别 → 知识点 + 思维导图 + 讲题引导 + 出题训练 |
| 综合分析 | 根据天赋+年级+成绩 → 丢分分析 + 提分方案 + 训练计划 |
| 对话历史 | 新建/查看/继续历史对话 |
| AI 模型 | 火山引擎豆包（或可替换为 DeepSeek） |
| 详情 | [学科答疑.md](功能模块文档/学科答疑.md) |

### 模块四：成长里程碑

| 项目 | 说明 |
|------|------|
| 功能 | 成长轨迹 + 荣誉体系 + 社交分享 |
| 荣誉级别 | 老板定义晋升路线和达成条件 |
| 精彩瞬间 | 自动记录：升段、连升两级、首次通关、训练满100天等 |
| 发圈文案 | 老板提供模板，一键复制分享 |
| 一键转发 | 公众号群发、微信分享（技术待学习） |
| 联动 | 打卡数据 + 测评结果 → 触发里程碑 |

## 五、用户流程

### 新用户

```
下载 App
  → 注册（填写年级/成绩等个人信息）
  → 天赋测评 → 获得天赋类型
  → 老学员自评 → 确定训练等级
  → 首页（四宫格）
      ├─ 天赋测试 → 查看报告 / 重新测评
      ├─ 今日训练 → 方案 → 视频 → 音频 → 打卡 → 报告
      ├─ 知识答题 → 语音/文字/拍照 → AI 个性化回答
      └─ 成长里程碑 → 荣誉 + 精彩瞬间 + 分享
```

### 老用户回访

```
登录 → 首页 → 今日训练方案 → 打卡
            → 知识答题 → 继续上次对话
            → 成长里程碑 → 查看新成就
```
