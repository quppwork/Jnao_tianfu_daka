# JNAO AI 产品规格

> 最后更新：2026-07-02
> 版本：v2.1
> 变更：训练模型从"主线 A→E"重构为"多技能并行 Tier 独立晋级"

---

## 一、产品定位

JNAO 是一个面向 K12 学生与家长的 **天赋成长综合平台**。

```
┌──────────┬──────────┬──────────┬──────────┐
│ 天赋测试  │ 今日训练  │ 知识答题  │ 成长里程碑 │
│ 发现天赋  │ 每日打卡  │ AI 辅导   │ 追踪成长  │
└──────────┴──────────┴──────────┴──────────┘
```

**闭环链路：** 测天赋 → 定阶位 → 每日多技能并行训练 → AI 答疑 → 看成长 → 社交分享

## 二、角色体系

```
┌────────────────────────────────────────────────────┐
│  C端（学生端）— 当前开发                              │
│  ┌──────────┬──────────┬──────────┬──────────┐    │
│  │ 天赋测试  │ 今日训练  │ 知识答题  │ 成长里程碑 │    │
│  └──────────┴──────────┴──────────┴──────────┘    │
├────────────────────────────────────────────────────┤
│  B端（导师端/家长端）— 后续开发                       │
│  ┌──────────────────────────────┐                 │
│  │  学习监督（提醒+打分+干预方案）│                 │
│  └──────────────────────────────┘                 │
└────────────────────────────────────────────────────┘
```

| 角色 | 说明 | 优先级 |
|------|------|--------|
| **学生** | 使用四个模块 | 🥇 当前 |
| **家长** | 与学生共用账户，可绑定多个学生 | 🥈 后续 |
| **导师（公司）** | 监督+打分+营销指导 | 🥈 后续 |

### 学生端（当前开发范围）

- 一个学生一个账户
- 注册时填写：年级、成绩水平、训练经历等（字段清单待老板确认）
- 首次使用：天赋测评 → 获得天赋类型 → **默认 Tier 1 初始状态**

### 家长端（后续）

- 与学生共用账户
- 一个家长可绑定多个学生（每多一个单独购买）
- 首页可切换子女
- 查看各子女的训练数据与报告

## 三、四大模块

### 模块一：天赋测试

| 项目 | 说明 |
|------|------|
| 功能 | AI 驱动的天赋测评 |
| 题目 | 35 道选择题（是/否），约 8 分钟 |
| 类型 | 成人版、儿童版 + 备用卷 |
| 输出 | 五维雷达图 + 情绪仪表 + 天赋解读 + 发展建议 |
| 天赋类型 | 思者 / 赢者 / 德者 / 行者 / 学者（+ 迷者，仅儿童） |
| 后端 | JNAO 外部 API（m.jnao.com）代理 |
| 详情 | [天赋测试.md](功能模块文档/天赋测试.md) |

### 模块二：今日训练（学员指导）🆕 v2.0

| 项目 | 说明 |
|------|------|
| 功能 | 根据天赋类型提供每日训练方案，**多技能并行训练** |
| 模型 | **技能独立 Tier 晋级**：每个技能单独判定连续3次达标→Tier+1；`overall_tier = min(所有技能 tier)` |
| 内容 | 音频训练 + 教练文案，按**时长公式**生成每日训练项 |
| 必修技能 | 超脑阅读(A)、影像追忆(B)、扫描速记(C)、极速运算(D)、极速学习(E) |
| 选修技能 | 精力恢复(>8h弹窗)、多元感知(可打可不打)、高效作业(不打卡不阻塞) |
| 排课 | 用户选时长 → 公式引擎展开技能组合 → 取 OSS 对应 stage/part 音频 |
| 打卡 | **顺序打卡**：必修项严格按序解锁；选修不阻塞后续 |
| 晋级 | **连续3次达标 → 技能 Tier+1**；不达标重置计数 |
| 后端 | `training_schedule_service.py` + `training_formula_engine.py` + `training_mastery.py` |
| 详情 | [今日训练.md](功能模块文档/今日训练.md) + [标准流程理解文档.md](标准流程理解文档.md) |

### 模块三：知识答题（学科答疑）

| 项目 | 说明 |
|------|------|
| 功能 | AI 一对一辅导，结合天赋类型个性化回答 |
| 输入 | 语音输入 + 文字输入 + 拍照发题 |
| 输出 | 文字回答 + 语音播报（火山引擎 TTS，张宇老师声音） |
| 个性化 | 根据天赋类型注入 AI 提示词（数学/语文/英语分科） |
| 学科 | 数学 / 语文 / 英语，自动识别 + 错频道提醒 |
| 会话 | 新建/继续/删除历史对话 |
| AI 模型 | 火山引擎豆包 |
| 详情 | [学科答疑.md](功能模块文档/学科答疑.md) |

### 模块四：成长里程碑

| 项目 | 说明 |
|------|------|
| 功能 | 成长轨迹 + 荣誉体系 + 社交分享 |
| 荣誉级别 | 老板定义晋升路线和达成条件 |
| 精彩瞬间 | 自动记录：升段、连升两级、首次通关、训练满100天等 |
| 发圈文案 | 老板提供模板，一键复制分享 |
| 一键转发 | 公众号群发、微信分享（技术待学习） |
| 联动 | 打卡数据 + 测评结果 → 触发里程碑 |

## 五、工作流程线

### 流程线一：天赋测试（🔗 完整可用）

```
Splash 启动页(3秒) → 首页
  → 点"天赋测试" → 大门页(选成人/孩子)
      ├─ 选孩子 → 年龄确认(满18?/未满18→提示) → 确认开始
      └─ 选成人 → 直接确认开始
  → 答题 35题
      ├─ 15秒倒计时(SVG环) → 超时后排到后面
      ├─ 选择"完全符合"/"有差异" → 随机反馈 + 里程碑提示
      ├─ 撤销上一题(5秒窗口)
      └─ 35种渐变背景
  → 完成页(弹出动画→文字→按钮)
  → POST /api/talent/report → 后端 → JNAO API(m.jnao.com) → 返回报告
  → 报告页(白色主题)
      ├─ Hero(天赋Logo+名称)
      ├─ 统计行 + 天赋特质 + 双重属性
      ├─ 五维雷达图SVG + 综合能力条形图
      ├─ 情绪仪表SVG + 状态标签
      ├─ 建议 + 三条黄金建议
      └─ 旧版报告链接
  → 返回首页 / 查看历史报告(localStorage)
```

### 流程线二：首页引导对话（🔗 完整可用）

```
首页 → 输入文字 → Enter发送 / 点发送按钮
  → POST /api/guide/chat → 豆包AI(doubao-seed-2-0-lite)
  → 显示回复
  → 🔊 语音播报(浏览器 SpeechSynthesis)
  → 🎤 语音输入(Web Speech API 实时转文字→自动发送)
  → Shift+Enter 换行
```

### 流程线三：今日训练（✅ 已实现）

```
首页 → 点"今日训练" → 校验天赋测评 → 每日方案卡
  → A方案训练(视频+音频) → 逐项打卡 → B方案解锁
  → B方案打卡 → 训练完成 → 成长里程碑联动
```

### 流程线四：知识答题（✅ 已实现）

```
首页 → 点"知识答题" → AI老师形象
  → 语音/文字/拍照提问 → 豆包个性化回答(注入天赋+分科提示词)
  → 对话历史(新建/继续/删除)
```

### 流程线五：成长里程碑（✅ 已实现）

```
首页 → 点"成长里程碑" → 荣誉徽章墙 + 成长时间线 + 一键分享
```
