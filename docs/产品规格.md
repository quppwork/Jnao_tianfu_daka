# JNAO AI 产品规格

> 最后更新：2026-06-21  
> 版本：v2.0 — 从单一评测平台升级为四大模块综合平台

---

## 一、产品定位

JNAO 是一个面向 K12 学生与家长的 **天赋成长综合平台**，包含四大核心功能：

```
┌──────────┬──────────┬──────────┬──────────┐
│ 天赋测试  │ 今日训练  │ 学科答疑  │ 成长里程碑 │
│ 发现天赋  │ 每日打卡  │ AI 辅导   │ 追踪成长  │
└──────────┴──────────┴──────────┴──────────┘
```

**一句话：** 测天赋 → 定方向 → 练能力 → 看成长，形成闭环。

---

## 二、四大模块

### 模块一：天赋测试

| 项目 | 说明 |
|------|------|
| 功能 | AI 驱动的天赋测评 |
| 题目 | 35 道选择题（是/否），约 8 分钟 |
| 类型 | 成人版（14岁以上）、儿童版（6-13岁） |
| 输出 | 雷达图 + 情绪分析 + 五大维度解读 + 发展建议 |
| 后端 | JNAO 外部 API 代理 |
| 状态 | ✅ **已实现**（前端 React，需重写为 Vue） |

### 模块二：今日训练（打卡系统）

| 项目 | 说明 |
|------|------|
| 功能 | 每日打卡训练，按方案顺序解锁 |
| 结构 | 训练 A → 打卡 → 解锁训练 B |
| 内容 | 视频训练 + 音频训练（通用标记，具体内容可配置） |
| 联动 | 打卡数据 → 成长里程碑 |
| 状态 | 🔨 **方案设计阶段** |

### 模块三：学科答疑

| 项目 | 说明 |
|------|------|
| 功能 | AI 一对一学科辅导 |
| 学科 | 数学、语文、英语、科学 + 自由提问 |
| 后端 | 代理 tianfu_rag 的 `/chat` 和 `/chat/stream` |
| 状态 | 🔒 **待开发** |

### 模块四：成长里程碑

| 项目 | 说明 |
|------|------|
| 功能 | 成长轨迹追踪 + 徽章系统 |
| 数据源 | 天赋测评结果 + 打卡记录 + 答疑记录 |
| 展示 | 时间线 + 徽章墙 |
| 状态 | 🔒 **待开发** |

## 四、用户流程

### 新用户

```
注册/登录 → 首页（四宫格）
              ├─ 天赋测试 → 选择类型 → 35题 → AI 报告
              ├─ 今日训练 → 方案 → 视频 → 音频 → 打卡
              ├─ 学科答疑 → 选学科 → AI 对话
              └─ 成长里程碑 → 时间线 + 徽章
```

### 老用户回访

```
登录 → 首页 → 查看今日训练方案 → 继续打卡
            → 查看成长轨迹变化
            → 重新测评（可选）
```
