## 二、页面状态机（6 个阶段）

整个天赋测试由 `TalentTest` 组件内的一个状态机驱动：

| 阶段 | phase 值 | 用户看到什么 | 用户操作 |
|------|---------|------------|---------|
| 1 | `door` | 欢迎语 + "孩子测试"/"成人测试"两个大按钮 | 点击选择类型 |
| 2 | `ageGate` | 年龄确认："已满18"/"未满18" | 仅孩子测试触发，点选确认 |
| 3 | `confirm` | 测试说明（35题、两选项、约10-15分钟）+ "准备好了"/"稍后再说" | 确认或返回 |
| 4 | `testing` | 题目卡片 + 倒计时 + 进度条 + AI提示 | 选择"完全符合"/"有差异" |
| 5 | `completed` | 撒花动画 + "35题完成" + "生成报告"按钮 | 点击生成报告 |
| 6 | `report` | 完整报告页（雷达图、情绪仪表、建议等） | 查看/重新测试/深度校准 |

## 三、各阶段详细设计

### 阶段 1：大门页 (door)

**UI 元素：**
- 顶部：欢迎语（新用户："首先我们来打开 Jnao 的大门，测试一下天赋吧！"；老用户："又来测试天赋了吗，欢迎"）
- 中间：两个按钮卡片，上下排列
  - "孩子测试" — 小孩图标 + 副标题"家长辅助完成 / 了解孩子的天赋"
  - "成人测试" — 成人图标 + 副标题"自行评估完成 / 探索内在潜能"

**关键状态：** `testType: "成人" | "孩子" | null`

---

### 阶段 2：年龄门控 (ageGate)

> 仅当选择"孩子测试"时进入；选"成人测试"直接跳到确认页。

**UI 元素：**
- "注意！请确认您的孩子是否满18岁"
- "已满18岁"（进入成人测试）
- "未满18岁"（进入孩子测试，显示家长辅助提示 2.2 秒后自动跳转）

---

### 阶段 3：确认页 (confirm)

**UI 元素：**
- "好的，{成人/孩子}测试。"
- "共 35 道题，每题两个选项：「完全符合」或「有差异」。根据实际情况选择即可，大约需要 10-15 分钟。"
- "准备好了吗？"
- "准备好了，开始吧"（进入测试）
- "稍后再说"（返回大门）

---

### 阶段 4：答题页 (testing)

**UI 布局（从上到下）：**

```
┌──────────────────────────────────────┐
│ [成人测试]              跳过→报告     │  顶部栏
├──────────────────────────────────────┤
│ ████████░░░░░░░░  5/35  14%         │  进度条
├──────────────────────────────────────┤
│                                      │
│         [上一题卡片（模糊，点击撤销）]  │  撤销区
│                                      │
│  ┌──────────────────────────────┐   │
│  │  第 5 题                     │   │
│  │  ⏱ 8s  ○══════════          │   │  倒计时环 + 进度条
│  │                              │   │
│  │  我喜欢尝试用新的方法来       │   │  题目标题
│  │  完成复杂的任务               │   │
│  │                              │   │
│  │  [完全符合]    [有差异]       │   │  选择按钮
│  └──────────────────────────────┘   │
│                                      │
│  🤖 AI提示：凭第一感觉选择就好～      │  AI 引导
└──────────────────────────────────────┘

+ 背景：每道题不同颜色主题（35 种渐变）
```

**核心逻辑：**

| 功能 | 说明 |
|------|------|
| 题目顺序 | 35 题 Fisher-Yates 随机打乱 |
| 倒计时 | 每道题 10 秒，最后 3 秒变红 |
| 超时处理 | 超时题目自动排到剩余题目的后半部分 |
| 撤销 | 点击上一题卡片 → 回退，5 秒内可重选 |
| 跳过 | 顶部"跳过→报告"直接跳到完成页 |
| 里程碑提示 | 第 5/10/15/20/25/30/35 题显示进度通知 |
| AI 提示 | 每题有预设的 AI 引导文案（来自 questionHints.ts） |

**答题编码：** `完全符合` → `1`，`有差异` → `0`，按题号 1-35 拼成 35 位字符串

---

### 阶段 5：完成页 (completed)

**UI 元素（按时间顺序自动显示）：**

| 时间 | 内容 |
|------|------|
| 0ms | 撒花/彩纸屑 Lottie 动画 |
| 1200ms | "35 题已完成" 淡入 |
| 1200ms | "AI 将为你生成专属天赋解读" 淡入 |
| 1800ms | "生成报告" 按钮弹簧动画弹出 |

**点击生成报告 →** POST `/api/talent/report` { answer, uid, type } → 进入报告页

**错误处理：** 请求失败显示红色错误信息，用户可重试

---

### 阶段 6：报告页 (report)

**十大区块（长滚动页面）：**

| # | 区块 | 内容 |
|---|------|------|
| 1 | Hero | 天赋 Logo + 名称（如"学者"）+ 类型简介 |
| 2 | 统计行 | 核心天赋值(分数) / 天赋类型(带颜色) / 能量状态 |
| 3 | 天赋特质 | A-E 五个属性，每个带等级/数值/描述 |
| 4 | 双重属性 | 可折叠 HTML 详解 |
| 5 | 天赋能力 | "想对你说的话" + 能力解读 |
| 6 | 综合能力 | **雷达图**（五维：创新力/领导力/公信力/执行力/协调力）+ 水平条形图 + 解读 |
| 7 | 当前状态 | **情绪仪表**（8 段式量规）+ 状态标签行 + 状态解读 |
| 8 | 给你的建议 | 事业建议 + 情感建议 |
| 9 | 三条黄金建议 | 编号列表 |
| 10 | 底部操作 | 记录ID/时间 + "重新测试" + "深度校准（备用卷）"（仅儿童版） |

**底部按钮：**
- "重新测试" → 回到大门页
- "深度校准（备用卷）" → 仅儿童测试可见，用备用 35 题再测一次

## 四、数据流

```
前端 User → 选择类型 → 答 35 题
                          │
                          ▼ encodeAnswers()
                  35 位 0/1 字符串
                          │
                          ▼ POST /api/talent/report
                  后端 jnao_submit()
                     → GET m.jnao.com/h5/adult/submitanswer
                     → 拿到 record_id
                  后端 jnao_get_report(record_id)
                     → 或本地模板回退
                          │
                          ▼ 返回 JnaoReportData
                  前端 ReportPage 渲染
                  同时存入 localStorage 历史记录
```

## 六、迁移到 Vue 的注意事项

| React 原实现 | Vue 替换方案 |
|---|---|
| `motion.div` (Framer Motion) | CSS transition / `<Transition>` |
| `AnimatePresence` | `<TransitionGroup>` |
| `useState` / `useRef` | `ref()` / `reactive()` |
| `sessionStorage` / `localStorage` | `uni.setStorageSync()` (UniApp API) |
| Lottie (`@lottiefiles/dotlottie-react`) | `lottie-web` 或 uni-app 插件 |
| SVG 内联（RadarChart/MoodDashboard） | 可直接复用，SVG 无框架依赖 |
| Tailwind CSS | UniApp 不支持 Tailwind，需改用 scoped style 或 uni.scss |
| antd-mobile | uView / uni-ui 替代 |
