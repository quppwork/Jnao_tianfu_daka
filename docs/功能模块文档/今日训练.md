## 一、系统定位

打卡系统是 JNAO 天赋成长平台的四大核心模块之一，承载用户"每日训练"功能。v2.0 重构后采用**多技能并行训练 + 独立 Tier 晋级**模型。

### 四大模块关系

```
┌──────────┬──────────┬──────────┬──────────┐
│ 天赋测试  │ 今日训练  │ 学科答疑  │ 成长里程碑 │
│  (测评)   │  (打卡)   │  (问答)   │  (追踪)   │
└──────────┴──────────┴──────────┴──────────┘
      ↑         ↑          ↑          ↑
      └─────────┴──────────┴──────────┘
                共用后端 API
                共用用户体系
```

## 二、v2.0 训练模型

### 2.1 技能分类

```
┌─────────────────────────────────────────────────┐
│              必修技能（5个）— 必须按序打卡            │
│  A. 超脑阅读  │  B. 影像追忆  │  C. 扫描速记        │
│  D. 极速运算  │  E. 极速学习                        │
├─────────────────────────────────────────────────┤
│              选修技能（3个）— 不阻塞后续              │
│  精力恢复(>8h弹窗) │ 多元感知(可打可不打) │ 高效作业(不打卡) │
└─────────────────────────────────────────────────┘
```

### 2.2 晋级机制

```
每个技能独立判定：
  打卡达标 → consecutive_pass += 1
  打卡不达标 → consecutive_pass = 0（严格连续，中断重置）
  consecutive_pass ≥ 3 → 技能 Tier + 1，计数重置

整体 Tier = min(所有活跃技能 Tier)  ← 最低原则
```

### 2.3 每日训练流程

```
用户选时长（20min / 40min / 1-2h / 2-3h / 3-4h / 4-5h）
  → 公式引擎展开技能组合（Tier 1: A + nB + C + 高效作业）
  → 取各技能当前 OSS (stage,part) 对应音频
  → 生成有序 TrainingItem 列表
  → 用户按序打卡（必修严格顺序，选修不阻塞）
  → 打卡卡片 → per-skill 判定达标/Tier晋级/OSS推进
```

## 四、状态流转（🆕 v2.0）

### 4.1 单技能 Tier 晋级

```
超脑阅读 Tier 1:
  打卡: 用时 2.5min, 字数 900
  → 900/2.5 = 360字/分 < 267字/分(800字/3min)?
  等等: 800字/3分钟 = 267字/分钟
  900/2.5 = 360字/分钟 ≥ 267 ✓ 达标
  → consecutive_pass: 0→1

第2天: 达标 → consecutive_pass: 1→2
第3天: 达标 → consecutive_pass: 2→3 → 超脑阅读 Tier 1→2!
  → 达标标准升级为 Tier 2: 2000字/5分钟
  → consecutive_pass 重置为 0

第4天: 不达标 → consecutive_pass: 0→0（重置）
```

### 4.2 整体 Tier

```
超脑阅读 Tier=2, 影像追忆 Tier=1, 扫描速记 Tier=1
→ overall_tier = min(2, 1, 1) = 1
→ 排课仍用 Tier 1 公式
```

## 六、打卡卡片字段（🆕 v2.0 扩展）

| 技能 | 卡片字段 | 判定方式 |
|------|---------|---------|
| 超脑阅读 | `{ name, time, wordCount }` | wordCount/time ≥ 阈值 wpm |
| 影像追忆 | `{ name, time, wordCount, accuracy }` | wordCount ≥ 阈值 + accuracy ≥ 阈值 |
| 扫描速记 | `{ name, wordCount, time, reverseRecite }` | wordCount/time ≥ 阈值 + reverseRecite=true |
| 极速运算 | `{ name, digits, correctCount, totalCount }` | 完成判定（后续可加正确率） |
| 极速学习 | `{ name, ... }` | 待量化 |
| 多元感知 | `{ name, card_count, accuracy_pct }` | 可选打卡 |

## 七、与其它模块的联动

| 联动方向 | 说明 |
|---------|------|
| **天赋测试 → 打卡** | 测评结果决定 OSS 内容池（五者不同音频） |
| **打卡 → 成长里程碑** | 连续打卡天数、Tier 晋级触发里程碑 |
| **打卡 → 学科答疑** | 训练中遇到的问题可跳转答疑模块 |
