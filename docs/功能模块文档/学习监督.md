# 学习监督（B端）

> 最后更新：2026-06-21  
> 状态：📋 已记录需求，暂不开发  
> 定位：导师端/家长端功能，非学生端

---

## 一、定位

学习监督是面向 **公司（导师）+ 家长** 的 B 端模块，不属于学生使用的四个核心模块。

当前四个模块（天赋测试、今日训练、知识答题、成长里程碑）全部是学生端。学习监督在导师端和家长端做。

---

## 二、需求记录

### 2.1 三方定时提醒

| 提醒类型 | 对象 | 说明 |
|---------|------|------|
| 定时训练提醒 | 家长 + 孩子 | 每日训练时间到了，提醒开始训练 |
| 定时异常提醒 | 导师（公司） | 学生训练数据异常（中断、退步） |
| 定时考试提醒 | 三方 | 考试时间通知 |
| 定时参会提醒 | 三方 | 会议/活动通知 |
| 定时销售提醒 | 导师 | 续费/升级提醒 |

**可能的实现方式：** 微信公众号模板消息 / 微信群发机器人（2人团队暂不会，后续学习）

---

### 2.2 训练执行度打分

由导师对学生的训练态度打分，六档：

| 分数 | 标签 | 描述 |
|------|------|------|
| 100% | 🔴 已透支 | 身体已透支，精神还要求进步 |
| 80% | 🟡 有余力 | 能完成任务，但还有余力学习 |
| 60% | 🔵 被动低效 | 做基本任务，被动的低效训练 |
| 40% | 🟤 逃避训练 | 不完成任务，不认真逃避训练 |
| 20% | ⚫️ 不配合 | 不完成任务，基本不配合训练 |
| 0% | ☠️ 严重不配合 | 不完成任务，严重不配合训练 |

**公式：** 态度分 = 提分方案 × 态度百分比

**执行位置：** 导师端页面

---

### 2.3 干预方案

根据不同对象生成不同指导内容：

| 对象 | 内容 |
|------|------|
| **公司（导师）** | 个性化营销指导：关键销售话术、产品推广话术、服务沟通技巧 |
| **家长** | 亲子沟通技巧：激励孩子、突破瓶颈、压力排解、生活习惯 |
| **学生（孩子）** | 阶段性提醒与警告：能力退步、错误习惯、学习态度、考前指导、心理健康 |

**注意：** 学生端干预 ≠ 知识答题。知识答题是学业辅导，学习监督的干预是行为/态度/心理层面。
